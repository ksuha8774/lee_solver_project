#реализация волнового алгоритма Ли для поиска кратчайшего пути в сетке с различными типами поверхности.

## Что реализовано

Программа `main.py` включает:

1. Загрузку сетки из CSV/TXT-файла.
2. Выбор начальной и целевой точки мышью.
3. Поиск кратчайшего пути.
4. Учёт разных типов поверхности:
   - `0` — обычная клетка;
   - `1` — препятствие;
   - `2` — трава;
   - `3` — песок;
   - `4` — сложная поверхность/вода.
5. Поддержку шагов:
   - `1` — по горизонтали и вертикали;
   - `sqrt(2)` — по диагонали;
   - `sqrt(5)` — нестандартные направления.
6. Запрет перепрыгивания через препятствия.
7. Запрет срезания углов при диагональном движении.
8. Сохранение сетки и найденного пути в SQL-хранилище SQLite.
9. Визуализацию результата в GUI.

## Как запустить

Нужен Python 3.10+.

```bash
python main.py
```

Дополнительные библиотеки устанавливать не нужно.

## Как пользоваться

1. Запустить `main.py`.
2. Можно сразу работать с демонстрационной сеткой или нажать **«Загрузить CSV»** и выбрать `sample_grid.csv`.
3. Нажать **«Выбрать старт»** и кликнуть по начальной клетке.
4. Нажать **«Выбрать цель»** и кликнуть по целевой клетке.
5. Нажать **«Запустить алгоритм»**.
6. Найденный путь будет подсвечен красным цветом.
7. Сетка и путь автоматически сохраняются в файл базы данных `lee_grid.db`.

## Структура базы данных

В SQLite создаются таблицы:

- `terrain_types` — типы поверхности;
- `grids` — сохранённые сетки;
- `cells` — клетки сетки;
- `paths` — найденные пути;
- `path_cells` — последовательность клеток найденного пути.

## Краткое описание алгоритма

Исходный алгоритм Ли обычно работает с одинаковой стоимостью переходов. В данной задаче стоимость переходов различается: обычный шаг стоит `1`, диагональный — `sqrt(2)`, нестандартный — `sqrt(5)`, а тип поверхности дополнительно увеличивает цену перехода.

Поэтому в программе используется обобщённая версия волнового алгоритма Ли — поиск через очередь с приоритетом, то есть алгоритм Дейкстры. Он гарантирует нахождение кратчайшего пути при неотрицательных весах переходов.
